# Blockchain

here are some tutorials and resources that can help you understand and use the combination of ERC721 and ERC20, as well as creating decentralized databases using blockchain technology:

    Using ERC721 and ERC20 together:

    "ERC721 & ERC20 Combined: Creating a Marketplace for NFTs with Fungible Tokens" by OpenZeppelin: https://docs.openzeppelin.com/learn/creating-a-marketplace-for-nfts-with-fungible-tokens
    "ERC721 & ERC20 Combined: Multi Token Marketplace" by ConsenSys: https://consensys.github.io/developers/tutorials/ethereum/erc721-erc20-combined-multitoken-marketplace/

    Creating decentralized databases using blockchain technology:

    "How to Build a Decentralized Database with Ethereum Blockchain" by freeCodeCamp: https://www.freecodecamp.org/news/how-to-build-a-decentralized-database-with-ethereum-blockchain-96d2476ce7f7/
    "Creating a Decentralized Database with IPFS and Ethereum" by Binance Academy: https://academy.binance.com/en/articles/what-is-ipfs-and-how-to-use-it
    "Building a Decentralized Database with Blockstack" by Blockstack: https://blockstack.org/docs/get-started/decentralized-database

These tutorials and resources should provide a good starting point for understanding and using the combination of ERC721 and ERC20, as well as creating decentralized databases using blockchain technology.
